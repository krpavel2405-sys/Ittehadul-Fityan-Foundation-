# ইত্তেহাদুল ফিত্ইয়ান ফাউন্ডেশন - ওয়েবসাইট

স্বাগতম আমাদের ওয়েবসাইট প্রজেক্টে! এটি ইত্তেহাদুল ফিত্ইয়ান ফাউন্ডেশনের জন্য একটি আধুনিক এবং প্রতিক্রিয়াশীল ওয়েবসাইট।

## প্রজেক্ট বর্ণনা

এই ওয়েবসাইটটি সংস্থার তথ্য, সেবা এবং যোগাযোগ তথ্য প্রদান করে। এটি নিম্নলিখিত বৈশিষ্ট্য সমৃদ্ধ:

### বৈশিষ্ট্যসমূহ

- **হোম পেজ**: আকর্ষণীয় হিরো সেকশন সহ স্বাগত পৃষ্ঠা
- **আমাদের সম্পর্কে**: সংস্থার বিস্তারিত তথ্য
- **সেবা**: আমাদের সমস্ত সেবার বিবরণ
- **যোগাযোগ ফর্ম**: দর্শকদের সাথে যোগাযোগের সুবিধা
- **রেসপন্সিভ ডিজাইন**: সকল ডিভাইসে নিখুঁতভাবে কাজ করে

## ফাইল কাঠামো

```
├── index.html       # মূল HTML ফাইল
├── styles.css      # CSS স্টাইলশীট
├── script.js       # JavaScript ফাইল
└── README.md       # এই ফাইল
```

## প্রযুক্তি স্ট্যাক

- **HTML5**: ওয়েব পেজের কাঠামো
- **CSS3**: আধুনিক স্টাইলিং এবং অ্যানিমেশন
- **JavaScript**: ইন্টারেক্টিভিটি এবং কার্যকারিতা

## ফিচার

### নেভিগেশন
- স্টিকি নেভিগেশন বার
- মসৃণ স্ক্রল আচরণ
- রেসপন্সিভ ডিজাইন

### হিরো সেকশন
- আকর্ষণীয় গ্র্যাডিয়েন্ট পটভূমি
- প্রভাবশালী শিরোনাম এবং বার্তা

### সেবা
- চারটি প্রধান সেবার বিবরণ
- আইকন এবং বর্ণনা সহ কার্ড ভিত্তিক লেআউট

### যোগাযোগ ফর্ম
- ব্যবহারকারী-বান্ধব যোগাযোগ ফর্ম
- ফর্ম ভ্যালিডেশন

## কীভাবে ব্যবহার করবেন

1. `index.html` ফাইলটি একটি ওয়েব ব্রাউজারে খুলুন
2. নেভিগেশন মেনু থেকে বিভিন্ন অংশ ব্রাউজ করুন
3. যোগাযোগ ফর্মে আপনার তথ্য পূরণ করুন
